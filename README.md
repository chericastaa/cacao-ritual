# Cacao Ritual

Sitio web de tienda de cacao ritual, desarrollado como trabajo práctico obligatorio para la materia de Desarrollo Web.

El proyecto es una adaptación temática de [tiendadecafe.com.ar](https://tiendadecafe.com.ar/), clonando su estructura, estilos y navegación con una identidad visual propia centrada en el cacao ceremonial de origen.

# Páginas

Archivo  Descripción 
`index.html` Página principal con hero, productos destacados y tutoriales 
`tienda.html` Catálogo completo de productos 
`tutoriales.html` Guías de preparación y recetas con cacao
`menu-tienda.html` Menú de la barra de bebidas y bocados
`franquicias.html` Información y formulario de franquicias
`trabaja-en-tdc.html` Puestos disponibles y formulario de postulación
`contacto.html` Datos de contacto y formulario de consultas

# Tecnologías

- HTML5 semántico
- CSS3 con Flexbox
- Google Fonts: Playfair Display + Raleway
- Font Awesome 6 para íconos
- Diseño responsivo con 4 breakpoints (1080px, 768px, 480px, 375px)
- Navegación mobile con CSS puro (sin JavaScript)
- Formularios funcionales vía FormSubmit
